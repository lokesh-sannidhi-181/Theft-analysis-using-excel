## 📌 Project Title
**Vehicle Theft Analysis in New Zealand**


## 📖 Project Overview
This project analyzes vehicle theft data across different regions of New Zealand to identify theft patterns based on vehicle type, model year, color, brand, location, and theft date.

The objective is to uncover actionable insights that can help law enforcement agencies, insurance companies, and vehicle owners better understand theft trends and take preventive measures.

## 🎯 Objectives
- Identify the most frequently stolen vehicle types.
- Determine which vehicle model years are targeted most often.
- Analyze the most stolen vehicle colors.
- Find regions with the highest number of vehicle thefts.
- Identify theft trends over time.
- Discover the most commonly stolen vehicle brands.

| Column    | Description     |

## 🔍 Key Insights
- Standard vehicle categories are more vulnerable to theft.
- Older vehicle models are targeted more frequently.
- silver vehicle colors show higher theft frequencies.
- Aukland regions experience substantially higher theft rates.
- Toyota brands are being targeted most.
